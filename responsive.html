For media query don't chess width of device, they are so many and coming up constantly with new width. So set break point as per suitable of design. For exampl if any element say navigation bar is looks nice upto mimimum width 500px then set break point 500px and change design and code for lower width. Same will be applicable for reverse condition. So your custom media query should be like - 

@media only screen and (max-width: 500px){/* For smaller/lower width than 500px */
	
}
@media only screen and (min-width: 500px){/* For biger/higher width than 500px */

}
